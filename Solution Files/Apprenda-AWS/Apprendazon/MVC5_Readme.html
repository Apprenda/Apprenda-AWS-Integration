Welcome to the Ignite UI site template! This starter application is meant to help you quickly get set up working with Ignite UI. You'll find examples of ways to interact with Ignite UI controls via plain JavaScript or through the ASP.NET MVC Helpers.

  
 

Speed up your development with the Ignite UI Page Designer. This drag and drop environment makes it easy for you to work with each of the Ignite UI controls along with build-in data sources as well as popular layout frameworks like Bootstrap.
 
 

Continue to learn as we have a wealth of resources available to you online in the form of Documentation, Code Samples, Application Samples and more.
